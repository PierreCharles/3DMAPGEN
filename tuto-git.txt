Pour la première utilisation, vous voudrez cloner ce qui est sur la forge, et le copier sur votre ordi.
Pour ça, on ouvre un terminal, on va dans le dossier dans lequel on veut qu'il y ai notre projet et on éxécute :

**Si vous avez pas les commandes de base de Linux sous Windows, désinstallez Git et réinstallez le en cochant
l'option Linux commandes (je sais plus exactement où c'est)**

git clone https://forge.clermont-universite.fr/git/cartes-tactiles

Cette commande va vous créer un dossier avec tout ce qu'il y a sur la branche principale (master) du projet.
A partir de là, vous avez en dur, sur votre PC, l'avancement du projet.



Récupérer les changements du projet :

git pull


Poster ces modifications sur la forge :

1- git status		//montre les fichiers que vous avez modifié
2- git pull		//On récupère d'abord les hypothéthiques changements de la forge
3- git commit -a		//valide les changements que vous avez fait
4- git push		//poste les modifs sur la forge



Si des nouveaux fichiers ont été créés (ex : ajout d'une classe, d'une image, etc..) :

La commande git status vous montrera quels sont les "untracked files"
Pour les ajouter au futur commit, on exécute :

git add <fichier qu'on veut ajouter>
OU
git add -A		// Dans le cas où trop de fichiers ont été créés

ensuite on peut faire git commit -a puis git push
